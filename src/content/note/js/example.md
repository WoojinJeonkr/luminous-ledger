# JavaScript Example

## Basic Syntax

```javascript
// This is a comment
console.log('Hello, World!');

// Variables
let name = 'John';
const age = 30;

// Functions
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrays
const fruits = ['apple', 'banana', 'orange'];

// Objects
const person = {
    name: 'Alice',
    age: 25,
    greet: function() {
        return `Hi, I'm ${this.name}`;
    }
};
```
